- Link component -
1) 특정 요소를 클릭하면, URL 경로가 바뀌면서 해당 경로로 이동하는 컴포넌트 
2) 클릭만 하면 이동하므로 특정 연산이 필요 없음 => a 태그와 동일한 기능

** a 태그와 link 태그는 페이지 로드 방식이 다름
a 태그 - 클릭할 때마다 페이지가 새로 불려짐 (외부 프로젝트를 불러올 때)
Link component - 렌더링할 부분만 렌더링되고 나머지는 유지
    => SPA 방식에서 사용
    => <Link /> : 프로젝트에서 페이지 전환 시

- useNavigate - Hook
1) 페이지 이동을 할 수 있게 해주는 함수를 반환
2) 반환되는 함수를 변수에 저장해놓고 변수의 인자로 설정한 패스값을 넘겨주면 해당 경로로 이동이 됨.
3) 조건이 필요한 요소에서 navigate 함수를 호출해서 이동 경로를 전달함
